Angular frontend scaffold for TinyURL

How to run:

1. Install dependencies
   npm install

2. Start dev server
   npm start

Notes:
- Default API base URL is set in `src/app/tiny.service.ts` to `https://localhost:7001`. Update if your backend runs on another port.
- This is a minimal scaffold. Use `ng` CLI (provided as devDependency) to build/serve.
